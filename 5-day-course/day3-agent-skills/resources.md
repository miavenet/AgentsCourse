# Day 3 — Agent Skills

Managing dynamic context and avoiding "context rot" with portable **Agent Skills** — directories structured around a central `SKILL.md` file (plus `scripts/`, `references/`, `assets/`). Progressive disclosure keeps system prompts lightweight, loading execution details and tools only on demand so a single agent can flex into hundreds of specialist roles.
