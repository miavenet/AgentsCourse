# Day 1 — Introduction to Agents & Vibe Coding

The transition from manual syntax coding to intent-driven "vibe coding" and disciplined "agentic engineering." How AI agents compress the traditional SDLC, and the "factory model" where developers act as system orchestrators designing evaluation, constraint, and context harnesses that safely guide autonomous execution.
