# Capstone — AI Agents: Intensive Vibe Coding Capstone Project

**Competition:** https://www.kaggle.com/competitions/vibecoding-agents-capstone-project

> ⚠️ As of 2026-07-02 the hackathon shows **~5 days to go** (deadline around July 7, 2026). Join soon if you want the certificate/badge/swag.

Put your course learning into action by building an agent that solves a real-world problem, helps people, or improves everyday living. The "Kaggriculture" capstone is a farming-simulation challenge where you build and deploy an autonomous agent to compete on a live leaderboard.

Completing the project earns official Kaggle certificates, profile badges, and a chance at exclusive swag.
